# 邮件查看布局修复任务

## 问题描述
邮件查看功能存在布局问题，主要表现在：
1. 邮件模态框内容可能溢出或滚动异常
2. 侧边栏邮件内容显示区域布局不够优化
3. 邮件HTML内容可能与扩展样式冲突
4. 响应式设计在小屏幕上可能有问题

## 修复计划
1. 优化popup.html中的邮件模态框布局和滚动
2. 改进sidebar.html中的邮件内容显示区域
3. 增强邮件内容的样式隔离和安全性
4. 优化响应式设计确保各种屏幕尺寸下正常显示
